# AeroSpace 快速上手指南

> AeroSpace v0.20.3 | macOS 平铺窗口管理器 | 基于你的 `~/.aerospace.toml` 配置

---

## 核心概念

```
┌─────────────────────────────────────────┐
│  Monitor (显示器)                        │
│  ┌──────────┐  ┌──────────┐             │
│  │Workspace 1│  │Workspace 2│  ...       │
│  │┌────┬────┐│  │          │             │
│  ││ W1 │ W2 ││  │          │             │
│  ││    │    ││  │          │             │
│  │└────┴────┘│  │          │             │
│  └──────────┘  └──────────┘             │
└─────────────────────────────────────────┘
```

- **Workspace（工作区）**：虚拟桌面，你配了 1-5
- **Tiling（平铺）**：窗口自动铺满屏幕，不重叠
- **Floating（浮动）**：传统自由窗口，手动拖拽

**所有快捷键以 `Alt`（⌥）为前缀**，不会和 ⌘ 系统快捷键冲突。

## 练习 1：焦点切换（最常用）

先打开 2-3 个应用窗口（如 Finder、Safari、Notes），让它们自动平铺。

```
⌥H  →  聚焦左边窗口
⌥L  →  聚焦右边窗口
⌥K  →  聚焦上方窗口
⌥J  →  聚焦下方窗口
```

> 试试：打开 Finder（⌘N），再打开 Safari，然后用 `⌥H` / `⌥L` 在它们之间切换。
> 你会看到窗口自动左右平铺，焦点切换不需要鼠标。

---

## 练习 2：移动窗口位置

```
⌥⇧H  →  把当前窗口移到左边
⌥⇧L  →  把当前窗口移到右边
⌥⇧K  →  把当前窗口移到上方
⌥⇧J  →  把当前窗口移到下方
```

> 试试：聚焦到右边的窗口，按 `⌥⇧H`，它会和左边的窗口交换位置。

---

## 练习 3：工作区

```
⌥1  →  切到工作区 1
⌥2  →  切到工作区 2
⌥3  →  切到工作区 3
⌥4  →  切到工作区 4
⌥5  →  切到工作区 5

⌥Tab →  在最近两个工作区之间快速切换
```

> 试试：
> 1. `⌥1` 确保在工作区 1，打开 Safari
> 2. `⌥2` 切到工作区 2，打开 Finder
> 3. `⌥Tab` 快速在 1 和 2 之间来回跳

## 练习 4：把窗口扔到其他工作区

```
⌥⇧1  →  把当前窗口移到工作区 1
⌥⇧2  →  把当前窗口移到工作区 2
⌥⇧3  →  把当前窗口移到工作区 3
```

> 试试：
> 1. 在工作区 1 打开 3 个窗口（太挤了）
> 2. 聚焦到一个不需要的窗口
> 3. `⌥⇧2` 把它扔到工作区 2
> 4. `⌥2` 切过去看看它在那里

## 练习 5：调整窗口大小

```
⌥=  →  当前窗口变大 (+50px)
⌥-  →  当前窗口变小 (-50px)
```

> 试试：两个窗口左右平铺时，聚焦左边的，连按几次 `⌥=`，看它变宽。

## 练习 6：布局切换

```
⌥/  →  在水平 ↔ 垂直平铺之间切换
⌥,  →  切换到手风琴模式（堆叠，⌥J/⌥K 翻页）
⌥F  →  当前窗口全屏（再按一次退出）
```

> 试试：
> 1. 打开 3 个窗口，默认水平排列
> 2. `⌥/` 切成垂直排列
> 3. `⌥,` 切成手风琴模式（只显示一个，⌥J/⌥K 切换）
> 4. `⌥/` 切回平铺

### 布局对比

```
 水平平铺 (tiles horizontal)    垂直平铺 (tiles vertical)    手风琴 (accordion)
┌──────┬──────┬──────┐       ┌──────────────────┐         ┌──────────────────┐
│      │      │      │       │       W1         │         │                  │
│  W1  │  W2  │  W3  │       ├──────────────────┤         │    W2 (可见)      │
│      │      │      │       │       W2         │         │                  │
│      │      │      │       ├──────────────────┤         │  ← ⌥J/⌥K 翻页 → │
│      │      │      │       │       W3         │         │                  │
└──────┴──────┴──────┘       └──────────────────┘         └──────────────────┘
```

## 练习 7：Service Mode（高级操作）

按 `⌥⇧;` 进入 service mode（注意状态变化），然后：

```
f    →  当前窗口在 floating ↔ tiling 之间切换（然后自动回 main mode）
r    →  重置当前工作区布局（flatten tree）
q    →  关闭除当前窗口外的所有窗口
Esc  →  退回 main mode（不做任何操作）
```

> 试试：
> 1. `⌥⇧;` 进入 service mode
> 2. `f` 让当前窗口变成 floating（可以自由拖拽了）
> 3. `⌥⇧;` → `f` 再切回 tiling

---

## 练习 8：Ghostty 专属操作

你的配置中 Ghostty 默认 floating（避免新 tab 导致 tiling 异常）。

```
# 打开 Ghostty → 默认是 floating 模式
# 想让它参与 tiling：
⌥⇧;  →  f    （切成 tiling）

# 想让它恢复 floating：
⌥⇧;  →  f    （再切回 floating）
```

## 日常工作流示例

### 开发场景

```
工作区 1：代码编辑器（全屏）           →  ⌥1
工作区 2：终端 + 浏览器（左右平铺）      →  ⌥2
工作区 3：文档/笔记                    →  ⌥3
工作区 4：通讯（Slack/微信）            →  ⌥4
工作区 5：音乐/杂项                    →  ⌥5
```

### 快速操作 Cheat Sheet

```
⌥ H/J/K/L          焦点：左/下/上/右
⌥⇧ H/J/K/L         移动：左/下/上/右
⌥ 1-5              切换工作区
⌥⇧ 1-3             移动窗口到工作区
⌥ Tab              上一个工作区
⌥ /                水平 ↔ 垂直
⌥ ,                手风琴模式
⌥ F                全屏
⌥ = / ⌥ -          放大 / 缩小
⌥⇧ ;               进入 Service Mode
  → f              floating ↔ tiling
  → r              重置布局
  → q              关闭其他窗口
  → Esc            取消
```

## CLI 调试命令

在终端中可以用 `aerospace` 命令查看状态：

```bash
# 列出所有工作区
aerospace list-workspaces --all

# 列出所有窗口
aerospace list-windows --all

# 列出当前工作区的窗口
aerospace list-windows --workspace focused

# 重载配置
aerospace reload-config

# 查看当前布局树
aerospace debug-windows
```
