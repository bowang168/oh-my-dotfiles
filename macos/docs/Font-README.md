## 场景推荐

### 写代码 / 终端
1. **MapleMono NF** — 首选，支持 Nerd Font 图标，连字清晰，`0Oo`、`1lI` 区分度高
2. **霞鹜文楷 Mono** — 需要在终端/编辑器中显示中文时使用
3. **InputMono** / **Menlo** — 干净的备选方案

### 长时间阅读（护眼）

| 语言 | 首选 | 备选 |
|------|------|------|
| 中文 | **霞鹜文楷 Screen** — 楷体笔画有节奏感，比黑体更护眼、更耐看 | 苹方 Regular（更干净但偏冷） |
| 英文 | **Charter** — 专为阅读设计，正文效果优于 InputSerif | New York、Georgia |
| 中英混排 | **霞鹜文楷** — 中英文搭配协调，阅读体验统一 | — |

### UI / 界面
- **苹方** — Apple 为 macOS 设计的中文系统字体，屏幕渲染最佳
- **InputSans** — 干净的英文无衬线字体，适合按钮、标签、标题

### 印刷 / 正式文档
- **汉仪楷体S** / **华文仿宋** — 传统中文字体风格，适合正式印刷品
- **Palatino** — 经典衬线字体，适合印刷英文文档

## 小贴士
- 正文阅读用 **Regular** 或 **Light** 字重，避免用 Bold
- 霞鹜文楷的 **Screen** 变体针对屏幕显示优化，日常阅读优先使用
- 中文字号建议 ≥ 14px，减轻视觉疲劳
- Charter 是 macOS 内置字体，无需安装，直接在应用中选择即可
